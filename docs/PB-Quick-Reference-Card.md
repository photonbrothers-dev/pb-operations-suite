## Understanding Priority Scores

Projects are scored 0-1000+ based on urgency:

| Factor | Points Added |
|--------|--------------|
| Days overdue | +1-3 per day (max ~100) |
| PE project | +30-50 bonus |
| High value ($50k+) | Up to +30 |

**Higher score = Schedule first**

## Daily Workflows

### Operations Manager Morning (15 min)
1. Open **Unified Command Center** → check red alerts
2. Open **Optimization Dashboard** → review top 10
3. Check **PE Dashboard** → milestone risks
4. Address urgent items

### Scheduler Weekly Planning (1 hour)
1. **Comprehensive Schedule** → RTB tab (26 ready)
2. **Master Scheduler v3** → check crew availability
3. Drag projects to calendar slots
4. Share schedule with crews

### Leadership Weekly Review (30 min)
1. **Pipeline Executive Summary** → overall health
2. **Revenue Forecast** → projections
3. **AHJ/Utility Tracker** → problem jurisdictions
